# v1.5.17 — 设计优化（首页单手操作 + 设计统一）

发布日期：2026-06-30
体验版包大小：__FULL__ ~642KB（编译 84 文件全部通过）

## 一、首页单手操作改造（用户主诉求）

- **背景**：原「发起球局 / 找场地」两个核心入口位于首页上方（hero 图下方 `.action-row`），单手握持时拇指难以触及屏幕上半区，不符合单手操作习惯。
- **改动**：
  - 移除顶部 `.action-row` 两个大按钮，避免重复。
  - 在首页**底部新增悬浮双按钮栏**（`fixed` 定位），「发起球局」「找场地」**等宽并排**，常驻拇指自然活动区，页面滚动时持续可见可点。
  - 主次视觉：发起球局＝墨绿实心（主操作）、找场地＝白底墨绿描边（次操作），带点击缩放反馈。
  - 适配底部安全区（`env(safe-area-inset-bottom)`），全面屏不被遮挡；列表底部留出安全距离，避免最后一张卡片被悬浮栏遮住。

## 二、设计统一（方案 A 小改动）

- **清理旧色**：全站残留旧色 `#00411A` 共 10 处（app.wxss 全局、game-detail、courses）已全部替换为运动风主墨绿 `#0B3D2E`（0 残留）。
- **统一设计规范**：在 `app.wxss` 追加统一组件样式（卡片 / 按钮 / 标签 / 空状态 / 加载态），供各页复用，作为后续设计基准。
- **管理后台规范对齐**：6 个管理页 + 后台首页的主卡片圆角统一为 24rpx、提交按钮统一为 999rpx 胶囊、头部 padding 顶部由 60rpx 收敛到 36rpx、页边距向前台 30rpx 对齐，整体观感与前台一致。
- **原则**：克制微调、不大动布局、零业务逻辑改动、风险最低。
