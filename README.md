# AceBook — Codex 版本存档

此仓库存放由 Codex 修改的 AceBook 小程序版本。

每个版本是一个完整的微信小程序源码目录，可直接导入微信开发者工具。

## 版本列表

| 版本 | 日期 | 标签 | 说明 |
|---|---|---|---|
| [v1.5.30](./v1.5.30) | 2026-07-08 | `v1.5.30` | 活动创建页UI重构 |

## 使用

```bash
# 克隆
git clone <仓库地址>

# 切换到某个版本
git checkout v1.5.30

# 把对应文件夹拖入微信开发者工具即可
```

## 工作流

每次 Codex 修改完成后：

1. 源码输出到本仓库
2. `git add` + `git commit` + `git tag`
3. 推送到 GitHub
