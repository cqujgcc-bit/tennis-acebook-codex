# v1.5.32 — 活动报名墙 + 排序过滤 + 历史活动折叠

> 日期：2026-07-11
> 基于 v1.5.31

## 改动

### 1. 报名墙（可看清谁报名了）

- **之前**：活动列表只显示一排小头像叠放，看不清谁报了名
- **之后**：每张活动卡片显示报名墙——头像+名字+标记
  - 👑 头像右下角显示「👑」角标（发起人）
  - 自己的名字加粗、显示绿色「我」徽标
  - 最多显示 4 人，超出显示 "+N"
  - 无人报名时显示"还没有人报名，快来占位"

### 2. 活动排序

- 活动中/待开始的活动按时间升序排列（即将到来的优先）
- 已结束的活动移入「历史活动」折叠区
- 历史活动按时间降序排列（最近的优先）
- 点击"展开 ▼"可查看历史活动列表（弱化显示，opacity 0.7）

### 3. 过滤规则

- 已结束（开始时间已过）且已结算/结算中的活动 → 历史活动
- 待开始 / 进行中未结算的 → 活跃活动列表
- 没有任何活动时显示空状态
